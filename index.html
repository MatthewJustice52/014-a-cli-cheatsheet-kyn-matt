Matt and Kenderlyn's Awesome Command line Cheatsheet

  • Command Line Interface (CLI) - a text interface navigated by typing commands instead of using the mouse.
  • Terminal - Mac version of command line interface
  • Bash - command line language
  • pwd - print working directory
  • hostname - computer's network name
  • mkdir - make directory
  • touch - create an empty file
  • cd - change directory
  • ls - list directory
  • rmdir - remove directory
  • cp - copy a file or directory
  • mv - move a file or directory
  • cat - print the whole file
  • sudo - super user root
